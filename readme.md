# furlify ☂️

Furl & Go!

## Getting Started

### Installation 💻

```
$ npm install -g furlify
```

### Usage ✈️

**Run in an existing project**

```
$ npx furlify chalk
```

You can also use multiple arguments separated with a space _(like `npm install`)_

```
$ npx furlify boxen chalk
```
